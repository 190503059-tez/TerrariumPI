Terrarium PI V. {{ version }}

TerrariumPI is started back in 2015 in order to keep my Daygecko happy. This piece of software is started as a home automation for reptiles, but now it can be used for indoor plants growing and paludariums.

This software is written in Python3 and uses the AdminLTE admin web interface.

For documentation about how to install and set it up, see https://theyosh.github.io/TerrariumPI

For bugs and feature requests please open a Github issue.

And there is a Matrix channel where you can try to ask questions and see if somebody can help you there: https://matrix.to/#/#terrariumpi:theyosh.nl
